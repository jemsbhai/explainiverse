# Evaluation Metrics

This directory contains the JavaScript/TypeScript implementations of the evaluation metrics from the Python `explainiverse` package.

## Roadmap

The following metrics are planned for porting:

### Faithfulness Metrics
1.  **PGI** (Prediction Gap on Important features)
2.  **PGU** (Prediction Gap on Unimportant features)
3.  **Comprehensiveness**
4.  **Sufficiency**
5.  **Faithfulness Correlation**
6.  **Faithfulness Estimate**
7.  **Monotonicity**
8.  **Monotonicity-Nguyen**
9.  **Pixel Flipping**
10. **Region Perturbation**
11. **Selectivity (AOPC)**
12. **Sensitivity-n**
13. **IROF**
14. **Infidelity**
15. **ROAD**

### Stability Metrics
16. **RIS** (Relative Input Stability)
17. **ROS** (Relative Output Stability)
18. **Lipschitz Estimate**
